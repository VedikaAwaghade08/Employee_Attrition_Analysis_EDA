# Employee Attrition Analysis (EDA)
**By Vedika Awaghade**

### Project Overview
This project performs Exploratory Data Analysis on the IBM HR dataset to identify the key factors that lead to employee turnover.

### Dataset
IBM HR Employee Attrition dataset containing 1470 employee records with demographic, job, and satisfaction-related features.

### Key Findings
* **Overtime:** Employees working overtime show a significantly higher rate of attrition.
* **Tenure:** Attrition is highest among employees with less than 2 years of tenure.
* **Job Level:** Level 1 (Junior) roles have the highest turnover rate.

### Strategic Recommendations
* **Targeted Retention:** Focus HR efforts on employees in their first 2 years.
* **Salary Benchmarking:** Re-evaluate compensation for Job Level 1 roles to stay competitive.
